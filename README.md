### Запуск приложения

Установите необходимые зависимости с помощью последовательности команд

```
cd ./frontend
npm i
cd ../backend
npm i
```

Запустите приложение последовательностью команд

```
cd ./frontend
npm run build
cd ../backend
npm start
```

Веб-приложение доступно по адресу:

```
http://localhost:3000
```
